## ***5 Estructura Basica***

*Sintaxis*
```html

    <header>
        <title>Mi titulo</title>
    </header>

        <h1>Titular 1</h1>
        <h2>Titular 2</h2>
        <h3>Titular 3</h3>
        <br>
        <h4>Titular 4</h4>
        <h5>Titular 5</h5>
        <h6>Titular 6</h6>
    </body>
</html>
```
